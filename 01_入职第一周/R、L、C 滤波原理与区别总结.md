一、相同点
电阻、电感、电容都可以参与滤波、降噪，核心作用都是：筛选不同频率的信号，去掉噪声，让电压 / 电流更平稳。
二、核心性格（最重要）
1. 电容 C
通交流，隔直流
对高频噪声：很通畅
对直流：相当于开路
作用：吃掉高频噪声，稳定电压
2. 电感 L
通直流，阻交流
对直流：很通畅
对高频噪声：拼命阻挡
作用：挡住高频干扰，DC-DC 储能滤波
3. 电阻 R
对所有频率一视同仁
不分直流、交流、高频、低频
都是消耗能量 → 变成热量
作用：搭配电容组成 RC 滤波
三、滤波方式对比
电容滤波
接法：并联在电源 / 信号与 GND 之间
逻辑：噪声通过电容流向地，干净电压留给芯片
电感滤波
接法：串联在供电线路中
逻辑：拦住高频噪声，不让它传到后级
电阻滤波（RC 滤波）
接法：电阻串联 + 电容并联到地
逻辑：电阻消耗能量，电容吸收噪声
适用：小信号、小电流，不适合主电源
四、快速对比表
表格
器件	对直流	对高频	滤波接法	典型用途
电容	隔断	导通	并联到地	去耦、稳压、滤高频
电感	导通	阻挡	串联在线路	电源滤波、DC-DC
电阻	消耗	消耗	配合电容	小信号 RC 滤波
五、记忆口诀
电容：高频走地，直流休息。电感：直流随便过，高频别想过。电阻：谁来都吃掉，不分高与低。
六、工程师实用结论
芯片旁边 0.1µF 电容：滤高频，必焊
DC-DC 电路功率电感：储能 + 滤波，必用
RC 滤波：只用于小信号，不做主电源滤波
电容和电感特性相反，经常配合使用（LC 滤波）
