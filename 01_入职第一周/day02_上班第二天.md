# 贴片元件BOM极简规范
### 核心模板
1.  **电容（CAP）**
CAP + 介质 + 容值 + 精度 + 耐压 + CH封装 + H厚度 + RO
例：CAP X5R 22uF +/-20% 25V CH0805 H1.45mm RO

2.  **电阻（RES）**
RES MF + 阻值 + 精度 + 功率 + CH封装 + H厚度 + RO
例：RES MF 0R +/-5% 1/16W CH0402 H0.4mm RO

3.  **NTC**
RES NTC + 阻值 + 精度 + B值 + CH封装 + H厚度 + RO
例：RES NTC 10K +/-1% B=3435 CH0603 H0.55mm RO

### 关键对应
- 精度字母：F(±1%)、J(±5%)、K(±10%)、M(±20%)
- 封装-功率：0402(1/16W)、0603(1/10W)、0805(1/8W)
