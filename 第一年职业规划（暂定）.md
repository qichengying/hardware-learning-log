# 第一年职业规划（硬件工程师方向）
## 一、总体目标
1. 顺利完成**毕业 + 入职**，平稳从学生过渡到职场
2. 夯实硬件基础：元器件、原理图、PCB、BOM、测试
3. 成为团队**可靠、上手快、能独立干活**的硬件工程师
4. 建立稳定工作习惯：复盘、文档、流程、沟通

---

# 第一阶段：实习期（前3个月）
**特点：工作时间不多 + 学校事务 + 毕设 + 适应职场**

## 1. 职场适应
- 熟悉公司流程、部门分工、硬件开发流程
- 熟悉常用工具：AD/Cadence、万用表、示波器、BOM系统
- 学会基本沟通：汇报、提问、记录、反馈

## 2. 技能重点（轻量但扎实）
- 看懂**原理图基础**：电源、接口、滤波、保护电路
- 熟练**元器件选型**：电容、电阻、二极管、TVS、ESD、LDO
- 能独立完成：BOM核对、物料选型、器件替代、文档整理

## 3. 学校与毕设
- 固定每周**固定时间写毕设**，不拖延
- 实习期间**不熬夜、不硬扛**，保证身体和效率
- 目标：平稳毕业，无后顾之忧

## 4. 阶段成果
- 能独立处理基础硬件辅助工作
- 毕设进度正常
- 顺利拿到转正答辩资格

---

# 第二阶段：试用期（第4～6个月）
**特点：全职工作 + 快速成长 + 转正考核**

## 1. 工作能力目标
- 独立负责**小模块电路**：电源、接口、滤波、保护等
- 熟练阅读硬件文档：规格书、应用手册、测试报告
- 参与打样、贴片、焊接、基础调试

## 2. 专业深度
- 掌握：电源设计基础、时序、纹波、散热、ESD防护
- 看懂PCB基本规则：阻抗、线宽、地平面、屏蔽
- 能输出：测试记录、问题分析、简单改善建议

## 3. 软能力
- 学会**主动汇报进度**
- 学会**定位问题、复现问题、描述问题**
- 做事有闭环：任务→执行→记录→反馈

## 4. 阶段目标
- 顺利通过试用期
- 能独立承担小型模块任务

---

# 第三阶段：正式工作期（第7～12个月）
**特点：独立负责 + 深度提升 + 成为骨干新人**

## 1. 核心能力目标
- 独立负责**完整功能模块**：电源、通信接口、传感器等
- 参与原理图设计、PCB评审、打样、调试、量产导入
- 能处理：EMC/ESD、散热、可靠性、BOM优化、降成本

## 2. 技术深度
- 掌握基础电源拓扑：LDO、DCDC、限流、反接保护
- 看懂基带相关：串口、SPI、I2C、电源管理
- 能做基础故障定位：电压、纹波、时序、通信异常

## 3. 职业素养
- 做事规范、文档清晰
- 能和软件、结构、测试高效配合
- 有自己的**硬件知识库/笔记/模板**

## 4. 一年终点目标
- 成为团队**放心使用的硬件工程师**
- 能独立负责模块，可带小任务
- 具备第二年往**资深/基带/电源**方向深耕的基础

---

# 全年极简时间轴（方便你写总结）
- **1～3月（实习）**：适应职场 + 熟悉基础 + 搞定毕设
- **4～6月（试用）**：快速上手 + 独立干活 + 顺利转正
- **7～12月（正式）**：独立负责 + 技术深耕 + 形成竞争力
